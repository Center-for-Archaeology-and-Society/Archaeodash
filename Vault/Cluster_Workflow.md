# Cluster Workflow

## Summary
Supports n-cluster diagnostics, HCA, HDCA, k-means, and k-medoids on elements or derived ordination spaces.

## Implementation
- Server/UI: `R/clusterTab.R`
- Results mapped back by `rowid`; assignments can be written to dataset columns
